# ALIZZ | Personal Portfolio Website

![Project Status](https://img.shields.io/badge/status-active-success)
![License](https://img.shields.io/badge/license-MIT-blue)

A stunning, responsive, and interactive personal portfolio website built with **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**. This project features advanced CSS animations, a dark-themed aesthetic with red accents, and an integrated background audio player.

## üåü Features

* **Interactive UI/UX:**
    * **Animated Background:** Dynamic gradient shift with particle effects.
    * **Typing Effect:** dynamic text presentation in the hero section.
    * **Cursor Trail:** A custom trail effect that follows the mouse cursor.
    * **Orbit Animation:** Tech stack icons rotating around the profile avatar.
* **Audio Player:** Built-in background music with a mute/unmute toggle button.
* **Responsive Design:** Fully optimized for desktop, tablet, and mobile devices using Tailwind CSS.
* **Project Showcase:** Grid layout to display portfolio projects.
* **Code Editor Visual:** A creative "IDE-like" representation of skills in the About section.
* **Social Integration:** Direct links to WhatsApp, TikTok, GitHub, and Telegram.
